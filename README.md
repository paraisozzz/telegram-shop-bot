# Telegram бот-магазин: Заработок на нейросетях

Готовый бот для продажи 4 цифровых продуктов через Telegram Stars.

## Что умеет бот
- показывает товары по команде `/start` и `/shop`
- принимает оплату в Telegram Stars
- автоматически отправляет сообщение после успешной оплаты
- поддерживает команду `/paysupport`
- готов к деплою на Render

## Файлы проекта
- `app.py` — основной код бота
- `requirements.txt` — зависимости Python
- `render.yaml` — конфигурация для Render
- `.env.example` — пример переменных окружения

## Как загрузить на GitHub
1. Создайте новый репозиторий на GitHub.
2. Загрузите в него все файлы из этого архива.
3. Не добавляйте реальный токен бота в репозиторий.

## Как запустить на Render
1. Откройте Render.
2. Нажмите **New +** → **Blueprint**.
3. Подключите GitHub-репозиторий с этим проектом.
4. При создании сервиса добавьте переменную окружения `BOT_TOKEN`.
5. Запустите деплой.

## Важно
- В GitHub нельзя хранить реальный токен бота в открытом виде.
- В этом архиве токен специально не записан в код, чтобы не скомпрометировать доступ к вашему боту.
- На Render токен нужно добавить в переменные окружения вручную.

## Команды бота
- `/start`
- `/shop`
- `/support`
- `/paysupport`

## Изменение товаров
Товары можно поменять в переменной `PRODUCTS_JSON` в Render или прямо в `app.py`.
