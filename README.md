# Telegram-бот для продажи цифровых товаров

Готовый бот-магазин для Telegram-канала.

## Что умеет
- показывает каталог товаров;
- продаёт цифровые товары через Telegram Stars;
- принимает оплату в XTR;
- автоматически доставляет товар после оплаты;
- хранит покупки в SQLite;
- поддерживает `/paysupport` для обращений по оплате.

## Важно
Для цифровых товаров в Telegram внутри приложения нужно использовать **Telegram Stars** (`XTR`).

## Быстрый запуск
1. Создайте бота через `@BotFather`.
2. Установите Python 3.11+.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Скопируйте `.env.example` в `.env` и вставьте токен.
5. Отредактируйте `products.json`.
6. Запустите:
   ```bash
   python bot.py
   ```

## Как подключить к каналу
1. Создайте канал-витрину.
2. Опубликуйте пост с кнопкой-ссылкой на бота: `https://t.me/USERNAME_BOT?start=shop`
3. Продавайте товары в личке с ботом.
4. Если хотите выдавать доступ в закрытый канал после оплаты — вставьте invite link в `delivery_value`.

## Форматы выдачи товара
В `products.json` можно использовать:
- `text` — ссылка, инструкция, код доступа;
- `document_file_id` — файл из Telegram по `file_id`;
- `photo_file_id` — изображение по `file_id`.

## Как получить file_id
1. Отправьте файл своему боту или в тестовый чат.
2. Временно добавьте обработчик логирования `document.file_id`.
3. Скопируйте `file_id` в `delivery_value`.

## Платежная логика
- Бот отправляет invoice.
- Telegram присылает `pre_checkout_query`.
- Бот подтверждает оплату.
- После `successful_payment` бот выдаёт товар.
